== Descrição ==
Plugin gratuito para envio de mensagens WhatsApp via Evolution API integrado ao WooCommerce. Inclui envio único, por status, carrinho abandonado, envio em massa, templates e validação no checkout.

== Instalação ==
1. Faça upload da pasta do plugin para /wp-content/plugins/
2. Ative o plugin no painel do WordPress
3. Configure a conexão no menu "WhatsApp Evolution"
